# 每日开源精选 · 2026-05-28

**microsoft/Webwright** · AI Web Agent 框架

> 用一个终端取代一整套 Web Agent 框架 —— 让 AI 写 Playwright 代码控制浏览器，而不是逐步预测点击坐标。

**项目概况**
- 微软研究院 AI Frontiers Lab 发布（2026 年 5 月）
- 累计 ⭐ 30,716 · Python · Trendshift 日榜 #5
- 极简设计：~1000 行代码，3 个模块，单一 Agent 循环

**核心成绩**
- <font color="info">Online-Mind2Web 86.7%</font>，开源方案排名第一
- <font color="info">Odysseys 长任务 60.1%</font>，超越此前 SOTA 35%
- GPT-5.4 平均每任务仅需 <font color="info">$2.37</font>，且产出可复用 CLI 脚本

**给架构师的学习要点**

<font color="info">代码作为接口</font>：给 AI 高层抽象（写 Playwright 代码）远比给它 50 种原子工具调用更有效。对应微服务里"暴露业务 API 而非数据库行操作"的设计哲学。

<font color="info">极简 vs 过度工程化</font>：1000 行代码拿下两个 SOTA，再次验证——模型能力提升后，复杂 orchestration 层的必要性要定期重新评估。

<font color="info">可复用工件沉淀</font>：每次任务都产出参数化 CLI 工具，能力可积累。有工具库加持时，Qwen3.5-9B（9B 小模型）也能达到 66.2%，降本提效的关键思路。

<font color="info">自我验证闭环</font>：强制在 done 之前走 Self-Reflection Gate，不接受 AI 自我声明，只接受可观察状态。生产 Agent 系统必备的工程约束。
